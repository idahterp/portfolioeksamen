Projekter Om mig

Website

I Tema 2 grundlæggende web lærte vi om generel HTML og CSS. Disse færdigheder brugte vi til først at lave mobilsite, for derefter lav en respontiv website som også fungerede som studiestarts prøve. Derudover blev vi introduceret til de grundlæggende færdigheder i grafisk opsætning af tekst og billeder i Figma, som vi senere brugte i Tema 3 og 5.

I website opgaven skulle vi vise at vi mestre de grundlæggende HTML og CSS. Vi skulle kode et site ud fra udleveret wireframes, layoutdiagram samt indhold (billede og tekst) til at lave et færdigt responsivt design.

Metode

Jeg opbyggede og stylede websitet med udleveret indhold. Dette gjorde jeg ved brugt af HTML til at strukturen af indholdet og CSS til at lave layout ud fra udleveret wireframes og layoutdiagram. Wireframes og layoutdiagram er med til at hjælpe programmøren til hvordan sitet skal opbygges, og mere overskueligt når man skal gå fra prototype til kodet site.

HTML bliver brugt til skruktur af tekst og billeder, igennem HTML sectioner, div og classes kan man bruge CSS til grid, strørrelse mm. Ved brug af CSS definerede jeg grid, layout samt design med farve, font-family samt media quries for at gøre websitet responsivt.

I det efterfølegende projekter har jeg videre brugt HTML og CSS, og udvidet mine færdigheder. Senere i Tema 4 i samspil med JavaScript.
