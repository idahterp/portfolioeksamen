Projekter Om mig

Website

Tema

I Tema 2 grundlæggende web lærte vi om generel HTML og CSS. Disse færdigheder brugte vi til først at lave mobilsite, for derefter lav en respontiv website som også fungerede som studiestarts prøve. Derudover blev vi introduceret til de grundlæggende færdigheder i grafisk opsætning af tekst og billeder i Figma, som vi brugte i tema 3.

Opgave

I website opgaven fik jeg udleveret wireframes, layoutdiagram samt indhold (billede og tekst) til at lave et færdigt responsivt design.

Metode

Jeg opbyggede og stylede websitet med udleveret indhold. Dette gjorde jeg ved brugt af HTML til indholdet og CSS til at lave layout ud fra udleveret wireframes og layoutdiagram. Dette gjorde jeg igennem HTML sectioner, div og classes. Ved brug af CSS definerede jeg grid, layout samt design med farve og font-family.
